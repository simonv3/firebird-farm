# Leaflet Web App

This is a simple web application that utilizes Leaflet.js to display a map with specific locations outlined on a small plot of land. The application features pop-up menus that showcase photos of the land when the markers are clicked.

## Project Structure

- **public/**: Contains static files for the application.
  - **index.html**: The main HTML file that loads the application.
  - **styles/**: Contains CSS files for styling the application.
    - **style.css**: The main stylesheet for the application.
  - **images/**: Contains images used in the application.
    - **placeholder.jpg**: A placeholder image for the pop-up menus.

- **src/**: Contains the source code for the application.
  - **app.js**: The main JavaScript file that initializes the Leaflet map and handles marker interactions.

- **package.json**: Contains metadata about the project and its dependencies.

## Getting Started

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your web browser to view the application.

## Dependencies

This project uses Leaflet.js for mapping functionality. Ensure you have an internet connection to load the Leaflet library from a CDN.

## Features

- Interactive map with outlined locations.
- Pop-up menus displaying images related to the locations.
- Responsive design for various screen sizes.
